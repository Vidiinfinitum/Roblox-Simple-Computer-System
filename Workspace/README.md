## Workspace

The **Workspace** module represents the computer's desktop environment and contains the majority of the system's functionality.

It provides several user interactions, including:

* Changing the desktop wallpaper.
* Customizing the user icon.
* Selecting the background music.
* Logging out of the current user session.
* Shutting down the computer.

In addition, the Workspace is responsible for launching the computer's **Game** application, which is currently the only executable available.

The system was designed with a modular architecture, making it straightforward to expand in the future. New desktop applications can be added with minimal changes to the existing codebase, allowing the computer to evolve without requiring major architectural modifications.
