### StartController

The **StartController.lua** module is the main controller for all interactions triggered by the **Start** button.

The customization system is divided into three modules:

* **Musics.lua** – Handles the selection of background music.
* **Wallpapers.lua** – Manages desktop wallpaper customization.
* **UserIcons.lua** – Handles user icon customization.

All three modules rely on **UIFunctions.lua**, which contains shared user interface utilities used throughout the customization system.
