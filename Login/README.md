## Login

After the computer finishes booting, the player is presented with the login screen.

From this interface, the player can either:

* **Start the user session** by clicking on the user profile, which loads the computer's desktop.
* **Shut down the computer** by clicking in the turn off button, triggering the shutdown sequence.

The login screen serves as the entry point to the desktop environment while providing a simple and intuitive interaction flow.
